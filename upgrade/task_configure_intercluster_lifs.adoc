= クラスタ間 LIF を設定しています
:icons: font
:imagesdir: ../media/


[role="lead"]
MetroCluster パートナークラスタ間の通信に使用するポートにクラスタ間 LIF を作成する必要があります。専用のポートを使用することも、データトラフィック用を兼ねたポートを使用することもできます。
