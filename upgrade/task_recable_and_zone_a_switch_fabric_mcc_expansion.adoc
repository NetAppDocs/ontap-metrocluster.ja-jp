= 新しいノードのスイッチファブリックの再接続とゾーニング
:icons: font
:imagesdir: ../media/


[role="lead"]
MetroCluster 構成にノードを追加する場合は、ケーブル接続を変更してから RCF ファイルを実行し、ファブリックのゾーニングを再定義する必要があります。

このタスクはスイッチファブリックごとに行う必要があります。一度に 1 つのファブリックに対して実行します。
