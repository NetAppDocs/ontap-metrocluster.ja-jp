= MetroCluster FC 構成へのストレージのホットアド
:icons: font
:imagesdir: ../media/


[role="lead"]
システムを停止せずに MetroCluster FC 構成にストレージを追加することができます。使用する手順は、 MetroCluster FC 構成の種類、および 1 台のディスクシェルフを既存のスタックに追加するか、またはスタック全体を追加するかによって異なります。
