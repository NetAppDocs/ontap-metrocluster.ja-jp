= ONTAP での MetroCluster ソフトウェアの設定
:icons: font
:imagesdir: ../media/


[role="lead"]
MetroCluster 構成の各ノードは、ノードレベルの設定や 2 つのサイトへのノードの設定を含めて、 ONTAP で設定する必要があります。また、 2 つのサイト間に MetroCluster 関係を実装する必要があります。
