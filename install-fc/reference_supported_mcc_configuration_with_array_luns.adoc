= アレイ LUN を使用する MetroCluster 構成がサポートされます
:icons: font
:imagesdir: ../media/


[role="lead"]
アレイ LUN を使用して MetroCluster 構成をセットアップできます。ストレッチ構成とファブリック接続構成の両方がサポートされます。AFF システムはアレイ LUN ではサポートされません。
