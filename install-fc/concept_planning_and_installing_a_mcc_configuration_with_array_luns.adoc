= アレイ LUN を使用する MetroCluster 構成を計画および設置する
:icons: font
:imagesdir: ../media/


[role="lead"]
MetroCluster 構成でアレイ LUN を使用している場合は、設置の計画を立て、その構成に固有な手順に従う必要があります。MetroCluster 構成は、アレイ LUN とネイティブディスクシェルフを混在させてセットアップするか、アレイ LUN のみを使用してセットアップできます。
