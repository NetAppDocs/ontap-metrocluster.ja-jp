= 7-Mode と clustered MetroCluster の間で共有するためのスイッチファブリックの設定
:icons: font
:imagesdir: ../media/


[role="lead"]
既存の 7-Mode ファブリック MetroCluster と新しい MetroCluster 構成との間でスイッチファブリックを共有するには、特定のゾーニングやその他の設定を行う必要があります。

このタスクは、両方のスイッチファブリックで一度に 1 つずつ実行する必要があります。
