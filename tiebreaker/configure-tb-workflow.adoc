= MetroCluster Tiebreaker 構成ワークフロー
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Tiebreakerソフトウェアをインストールした後、MetroCluster構成を追加し、SNMP設定を構成して、MetroCluster環境の監視を開始できます。

.このワークフローについて
このワークフローを使用すると、インストール後にTiebreakerソフトウェアを設定できます。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1つ"]link:concept_configuring_the_tiebreaker_software.html["Tiebreaker ソフトウェアを設定します"]
[role="quick-margin-para"]
MetroCluster構成をTiebreakerソフトウェアに追加し、証明書をインポートして、監視設定を構成します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["2 つ"]link:task_configure_snmp_setts_for_tiebreaker_software.html["MetroCluster TiebreakerソフトウェアのSNMP設定を構成します"]
[role="quick-margin-para"]
TiebreakerソフトウェアがSNMP通知をMetroClusterイベントに送信できるようにSNMP設定を構成します。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["3つ"]link:concept_monitoring_the_mcc_configuration.html["MetroCluster の設定を監視する"]
[role="quick-margin-para"]
Tiebreaker ソフトウェアを使用して MetroCluster 構成のステータスを監視し、SNMP イベントを表示して、AutoSupport 設定を管理します。
